# job-listing

A job listing UI with description page included

## To run the page follow the following instructions

- Clone this repo

```bash
git clone https://github.com/biniyamNegasa/job-listing.git
```

- Get into the folder

```bash
cd job-listing
```

- run the following

```bash
npm install && npm run dev
```
